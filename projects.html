Script d'installation d'Arch Linux

Le but initial était d'apprendre à installer un système Arch Linux complet sans utiliser l'outil archinstall. Lors de l'apprentissage, j'ai édité un fichier pour marquer les étapes et les commandes. Au final, j'ai décidé de le compléter pour en faire un script avec post-installation. Il installe donc également toutes les applications ainsi que ma configuration via GitHub. Voici les spécifications de l'installation :

Système de fichiers : btrfs

Chiffrement : LUKS

DE/WM : Swayfx

Système d'amorçage : systemd-boot

Script de backup

Je suis très adepte des scripts Bash. Celui que j'utilise le plus souvent est un petit script pour sauvegarder toutes les données que j'ai choisies sur un disque dur externe. Je branche le disque dur et lance le script. Ensuite, il s'occupe de monter le disque et de faire une sauvegarde ou une restauration en fonction du paramètre. Enfin, il démonte le disque et envoie une notification indiquant que le programme s'est bien déroulé. La commande rsync est utilisée pour synchroniser les données. Cela me permet de sauvegarder rapidement et régulièrement toutes mes données.

Capteur de température avec Arduino

J'ai utilisé un Arduino accompagné d'un capteur de température et d'humidité, un écran LCD et un lecteur de carte SD. Le but était d'analyser les niveaux de température et d'humidité et de les enregistrer sur la carte SD afin de faire des statistiques et de vérifier si l'environnement est correct.

Créer mon propre serveur

Créer mon propre serveur est un nouveau projet qui m'est venu à l'esprit. L'idée finale est de déployer un serveur dans ma future maison qui offrira plusieurs applications utiles. Je pense notamment à Wireguard, Adguard, Vaultwarden, Syncthing, Nextcloud, et bien d'autres. C'est un gros projet, mais également très intéressant. Cependant, vu l'importance des applications, il faut prendre le temps de faire énormément de tests et de progresser petit à petit. Le tout tournera probablement à l'aide de Docker afin de facilité de le déploiement.

Script d'installation d'Arch linux

Le but de base était d'apprendre à installer un système arch linux complet sans utiliser l'outil archinstall. Lors de l'apprentissage, j'ai édité un fichier pour marquer les étapes et les commandes. Au final, j'ai décidé de le ccompléter pour en faire un script avec post-installation. Il installe donc également toutes les applications mais également toutes ma config via github. Voici les spécifications de l'installation.

  • Système de fichier : btrfs
  • Chiffrement avec luks
  • DE/WM : swayfx
  • Système d'amorçage : systemd-boot

Capteur de température avec arduino

J'ai utilisé un arduino accompagné d'un capteur de température et d'humidité, un écran lcd et un lecteur de carte SD. Le but était d'analyser les niveaux de température et d'humidité et des les enregistrer dans la carte SD afin de faire des statistiques et de voir si l'envirronement est correct.
